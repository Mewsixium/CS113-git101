array คือ กลุ่มตัวแปรที่ถูกเก็บอยู่รวมกันโดยตัวแปรต้องเป็นประเภทเดียวกัน
โดยเมื่อเรียกข้อมูล ข้อมูลแต่ละตัวจะมีเบอร์ประจำข้อมูล เบอร์นั้นเป็นตัวบอกตำแหน่งของข้อมูลว่าอยู่ตำแหน่งใดของ array
การประกาศ int A[10] ; โดยจะเริ่มค่าจากตำแหน่ง 0 ไปจนถึง ตำแหน่งที่ 9
ขนาดของ array ควรเป็น ค่า constant จะไม่สามารถเลี่ยนแปลงตลอดการทำงานของโปรแกรม
#define arraysize 5 เปลี่ยนอักษรทั้งหมดในโปรแกรมก่อนคอมไพล์
การส่งค่าตัวแปรในarray เป็นการส่งแบบ pass by reference
การ pass by value เป็นการส่งตำแหน่งของข้อมูลจากผู้เรียกไปให้ฟังชั่น เมื่อเปลี่ยนค่าตัวแปรนั้นในฟังก์ชั่น ข้อมูลต้นตำหรับจพถูกเปลี่ยนด้วย
ตัวแปรหนึ่งตัวกินเนื้อที่ข้อมูลตามชนิดของข้อมุล
array 2D
เป็นการสร้างตารางแบบสองชั้น
A[i][j] โดย i จะเป็นตัวกำหนดจำนวนของแถวแนวนอน (row)
            j จะเป็นตัวกำหนดจำนวนของแถวแนวตั้ง (column)
