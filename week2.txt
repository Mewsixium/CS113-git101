week2 ชนิดตัวแปร 
ต้องประกาศตัวแปรพร้อมกำหนดชนิดข้อมูลให้กับตัวแปรด้วย
int คือ จำนวนเต็ม
float คือ จำนวนจริง
char คือ ตัวอักขระ
double คือ สามารถ เก็บจำนวนจริงที่มีจำนวนมากๆได้
long คือ เพิ่มขนาดการเด็บข้อมูล
ีืunsigned คือ เป็นการเพิ่มการเก็บข้อมูลโดยไม่เก็บค่าเครื่องหมาย
สามารถตรวจสอบขนาดของชนิดแต่ละตัวได้ด้วยคำสั่ง sizeof(ตัวแปร)
ค่าคงที่ข้องเลขฐาน 2 จะขึ้นต้นด้วย 0b
ค่าคงที่ข้องเลขฐาน 8 จะขึ้นต้นด้วย 0 (%o)
ค่าคงที่ข้องเลขฐาน 16 จะขึ้นต้นด้วย 0x (%x / %X)
\0 เป็น Null character
const สามารถใช้กับตัวแปรอาเรย์ เพื่อไม่ให้ข้อมูลภายในอาเรย์ถูกเปลี่ยนแปลงได้
c = getchar() รับข้อมูลเป็น สตริง จาก standard input
putchar(c) ส่งอักขระที่อยู่ในตัวแปร c ไปยัง standard output
EOF ค่าคงที่ที่ถูกกำหนดใน stdio.h บอกถึง end of file
+ - * / เป้นตัวดพเนินการทางคณิตศาสตร์
% ไม่สามารถใช้กับ float หรือ duoble ได้
ผลหารจะเป็นจำนวนเต็มถ้าตัวหารและตัวถูกหารเป็นจำนวนเต็ม
ตัวดำเนินการก่อน / * % + -
relational operators > >= < <= == !=
logical operators && || ! 
คำสั่งกำไหนดเงื่อนไข if() ,else if() ,else
array จะเก็บข้อมูลชนิดเดียวกันเท่านั้นอยู่ใน array 
float ไม่สามารถใช้เป็น subscript ใน array ได้
ตัวอักขระเป็นจำนวนเต็มสามาถถูกแปลงไปมาระหว่าง int กับ char ได้
false -->0 ,true -->1
